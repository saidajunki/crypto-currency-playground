# Cryptocurrency Playground

暗号通貨・ブロックチェーンの仕組みを学習するための教育用シミュレーション環境です。

## 概要

このプロジェクトは、Bitcoinなどの暗号通貨の基本的な仕組みをTypeScriptで再現し、
ローカル環境で実際に動かしながら学習できるplayground環境を提供します。

## 主な機能

- **ブロック**: トランザクションをまとめるデータ構造
- **ブロックチェーン**: ブロックを連鎖させて改ざんを検知
- **トランザクション**: デジタル署名付きの送金データ
- **ウォレット**: 公開鍵/秘密鍵ペアの管理
- **マイニング**: Proof of Workによるブロック生成
- **CLI**: コマンドラインからの操作

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発モードで実行
npm run dev

# ビルド
npm run build

# テスト実行
npm test
```

## 使い方

```bash
# CLIを起動
npm run dev

# 利用可能なコマンド
help                    # コマンド一覧を表示
wallet create <name>    # 新しいウォレットを作成
wallet list             # ウォレット一覧を表示
wallet balance <name>   # 残高を確認
send <from> <to> <amt>  # 送金
mine <wallet>           # マイニングを実行
chain                   # ブロックチェーンを表示
pending                 # 保留中のトランザクションを表示
```

## 学習ポイント

1. **ハッシュ関数**: データの改ざん検知
2. **公開鍵暗号**: デジタル署名による認証
3. **Proof of Work**: 分散合意の仕組み
4. **チェーン構造**: 不変性の実現方法

## 注意事項

- これは教育目的の実装であり、実際の暗号通貨として使用することはできません
- 難易度は低く設定されており、ローカルPCで快適に動作します
